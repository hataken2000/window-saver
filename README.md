# Window Saver

Chromeのウィンドウ状態（タブ・タブグループ）を名前をつけて保存・復元できる拡張機能です。

---

## インストール方法

1. このリポジトリを ZIP でダウンロードして解凍する（または `git clone`）
2. Chrome で `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. 解凍したフォルダを選択する

---

## 使い方

### セッションを保存する
1. 保存したいウィンドウをアクティブにする
2. 拡張アイコンをクリックしてポップアップを開く
3. 名前を入力（省略すると日時が自動入力）して「保存」をクリック

### セッションを復元する
1. ポップアップを開く
2. 一覧から復元したいセッションの「復元」をクリック
3. 新しいウィンドウで開く

### セッション名を変更する
- セッション名をクリックして編集、Enter で確定

### セッションを削除する
- セッション右端の「✕」をクリック

---

## エクスポート／インポート

**拡張を削除するとデータはすべて消えます。** 削除前に必ずエクスポートしてください。

- **エクスポート**：ポップアップの「エクスポート」ボタン → JSON ファイルをダウンロード
- **インポート**：「インポート」ボタン → 保存した JSON ファイルを選択
  - 既存セッションとマージされます（重複はスキップ）

---

## 他の拡張・Webページとの連携

`externally_connectable` に対応しています。別の拡張や許可済み Web ページから以下のメッセージで操作できます。

```js
const WINDOW_SAVER_ID = "window-saverのExtension ID"; // chrome://extensions で確認

// セッション一覧を取得
chrome.runtime.sendMessage(WINDOW_SAVER_ID, { type: "GET_SESSIONS" }, sessions => {
  console.log(sessions);
});

// セッションを復元
chrome.runtime.sendMessage(WINDOW_SAVER_ID, { type: "RESTORE_SESSION", session: sessions[0] }, res => {
  console.log(res); // { ok: true }
});
```

Web ページから呼び出す場合は `manifest.json` の `externally_connectable.matches` に対象 URL を追加してください。

---

## 注意事項

- `chrome://` や `devtools://` などの内部ページは保存対象外です
- ウィンドウを閉じると自動でセッションが更新されます
- 拡張を**再インストールするとデータは消えます**。事前にエクスポートを忘れずに
- デベロッパーモードの拡張はブラウザ起動時に警告が出ることがあります（正常です）
